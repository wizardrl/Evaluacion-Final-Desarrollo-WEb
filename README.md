# Impulso Social

Sitio informativo para la nueva **Licenciatura en Innovación Social** de la Facultad de Ciencias Sociales.

## Archivos

- `index.html`: página de inicio con carrusel de testimonios.
- `programa.html`: plan de estudios, tabla de datos y mapa de imagen interactivo.
- `contacto.html`: formulario con validación y mensajes de confirmación.
- `css/styles.css`: diseño Mobile First, adaptaciones responsivas y efectos visuales.
- `js/app.js`: año automático y validación del formulario.

Bootstrap 5 se incluye desde CDN para la navegación desplegable, carrusel y acordeón.

## Accesibilidad y usabilidad

- La estructura usa `header`, `nav`, `main`, `section`, `article`, `aside` y `footer`, con una jerarquía de encabezados coherente.
- Todas las páginas incluyen un enlace para saltar directamente al contenido, etiquetas `aria`, foco visible y navegación por teclado.
- Las imágenes informativas cuentan con texto alternativo; las decorativas se ocultan a lectores de pantalla mediante `aria-hidden`.
- La paleta combina texto oscuro y fondo claro, y botones con contraste alto. La información no depende solo del color.
- El formulario emplea `label`, tipos de entrada adecuados, validación nativa, foco en el primer error y una región `aria-live` para los mensajes de estado.
- El diseño comienza en pantallas pequeñas y se amplía con una media query. La tipografía y los espacios usan medidas flexibles para mantener la legibilidad.
